# 🔵 K-Means Clustering Algorithm

This project demonstrates the implementation of the **K-Means Clustering** algorithm using Python. It applies unsupervised machine learning techniques to group data into meaningful clusters.

## 📌 Project Overview
K-Means is an unsupervised learning algorithm used to partition data into **K distinct clusters** based on similarity. In this notebook, we load and explore the dataset, apply K-Means clustering, visualize clusters, analyze cluster centers, and evaluate clustering performance.

## 🧠 What is K-Means?
K-Means works by choosing the number of clusters (K), initializing centroids randomly, assigning data points to the nearest centroid, updating centroids based on assigned points, and repeating the process until convergence.
